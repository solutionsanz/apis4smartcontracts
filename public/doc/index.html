A) About GS Customizable drag and drop - top

GS Customizable drag and drop allows you to sort, add, edit and delete items on a list.

  • Set a initial order of the items - that will be displayed before the items are saved.
  • Set if you need to have handlers. Here we've added two options: generate handlers for all list items, or define your own handlers for a specific items.
  • Set if you want the Add option enabled - this will allow you to add new items to the list
  • Set if you want the Edit option enabled - this will allow you to edit all the items on the list
  • Set if you want the Delete option enabled - this will allow you to delete any item on the list

D) JavaScript - top

This product was developed using pure JavaScript. No extra libraries were used, meaning that no extra dependencies are required so that this script works. Anyway, this module is compatible and can be used along with most the known frameworks.

Callback functions

Event Callback Function Info
On list save onSave(e) This event triggers when the list is saved. In our example we are displaying a message.
AJAX Complete onAjaxComplete(e, data) This event triggers after the AJAX call on the server is complete
AJAX Failure onAjaxFailure(e) This event triggers if the AJAX call failed

Options

Option Description Accepts
save Set if the list will be saved. Boolean: true or false
Default: true
initialOrder Set up an initial order of the items that will be used before the Save button is pressed Array
Default: an empty array
sendToPhp Set if the list should be send to a php. This also needs the 'save' option set to true. Boolean: true or false
Default: false
sendToPhpUrl Set the AJAX call of the PHP that will receive the final object with the list. String
Default: empty string
useHandler Set if handlers should be used. Boolean: true or false
Default: false
generateHandler Set if you need default handlers on all list items. Boolean: true or false
Default: false
handlerClass Set if custom handlers are needed and add the handlers class. String
Default: empty string
editItem Set if the items should have the edit option. Boolean: true or false
Default: false
deleteItem Set if the items should have the delete option. Boolean: true or false
Default: false

F) Features - top

Here are the main features of the Gs Customizable drag and drop:

  • No jQuery!
  • Light weight JavaScript
  • Use AJAX to send to a PHP a JSON that will contain the final object of the list(with the items ordered, edited, deleted).
  • Fully customizable - you can disable/enable each option.
  • Customizable messages - you can manage the text for the Save option, or when an empty item is added to the list.
  • Sort the list by using drag and drop.
  • Addding new items to the list.
  • Editing/Deleting items from the list.
